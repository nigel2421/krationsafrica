# **App Name**: Kreations Kicks

## Core Features:

- Product Catalog & Display: Showcase new arrivals, product categories, and individual shoe cards with images, names, prices, and clear stock status (in stock/out of stock).
- Dynamic Shopping Cart: Allow users to add, view, update quantities, and remove items from a slide-out cart, dynamically calculating the total price.
- WhatsApp Order Generation: Capture delivery details (Name, Phone, Location) and generate a pre-formatted WhatsApp message with cart contents and order information to send to the store.
- Admin Authentication: Secure login for administrators using Firebase Authentication (Email/Password) to access the CMS dashboard.
- Product & Inventory Management: Admin dashboard functionality to perform CRUD operations (Create, Read, Update, Delete) on product listings and manage their stock status (In Stock, Out of Stock, Few Left).
- Reusable Media Library: Admin section to upload shoe images to Firebase Storage and select existing images when creating or editing products, optimizing storage and workflow.
- AI Product Description Tool: A generative AI tool for administrators to assist in creating concise and engaging product descriptions based on minimal input details.

## Style Guidelines:

- Color Palette: A sophisticated light scheme designed for modernity and clarity. The primary brand color is a deep indigo (#424266), offering a refined, elegant feel. The background is an almost-white cool grey (#FBFBFC) to ensure excellent readability and a minimalist canvas. A vibrant sky blue (#3AC8F3) acts as a high-contrast accent for interactive elements and key highlights.
- Font Recommendation: 'Inter' (sans-serif) for all text. Its contemporary, neutral, and highly readable characteristics make it suitable for both headlines and body text, ensuring a clean and consistent look across the application.
- Icon Style: Minimalist line icons for core actions and navigation, maintaining the sleek aesthetic. Prioritize clarity and intuitive recognition.
- Layout Structure: Employ spacious grid layouts with generous padding around product imagery and content blocks. This approach maximizes visual hierarchy, enhances breathability, and underscores the application's ultra-modern and clean design principles.
- Animation Style: Utilize subtle and purposeful micro-interactions with Framer Motion. Implement smooth transitions for navigation (e.g., between pages or opening the shopping cart) and hover effects on product cards to enhance user engagement without being distracting.
